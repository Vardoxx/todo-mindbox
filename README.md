# Развёртывание:

```js
  git clone https://github.com/Vardoxx/todo-mindbox.git
  cd todo-mindbox
  npm i && npm run start

  npm run test /*Запуск теста*/
```

# Используемые технологии:

- TS
- Tailwind
- React
- Vite
- Vitetest
